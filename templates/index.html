Project Overview


The purpose of this project is to predict if one’s income is more or less than 50K based on 10 given features. We used income classification dataset found on Kaggle.com. The dataset has 14 attributes and 1 target, we dropped and cleaned the data and remained 30162 rows of data and 10 features for our machine learning model.

To determine which machine learning model to use in prediction, we were applying three different machine learning models including support vector machines (SVMs), logistical regression and random forest and choosing one with the best performance for our model.

Machine Learning


Our goal is to accurately predict one’s income value is more or less than 50,000 USD. The result is demonstrated by the following major steps of machine learning.

1. Data Processing
  • Data source: Income Classification dataset from Kaggle.
  • Find and drop total of 2399 missing data and unnecessary columns. Remaining 30162 rows of data with 10 features and 1 target column (income).
  • Using Scikit-learn's One-Hot Encoding to fit and transform categorical variables.
  • Using Scikit-learn's StandardScaler module standardizes the data so that the mean of each feature is 0 and standard deviation is 1.
2. Selecting three different Machine Learning Models

In our analysis, the outcome of income prediction is a binary classification (>50K or <=50K). Here are the results of three different machine learning models which are commonly used in categorizing sample into groups.

  • Logistical Regression

    • Confusion matrix:

    Predicted Income <=50K Predicted Income >50K
    Actual Income <=50K 5231 411
    Actual Income >50K 798 1101
    Accuracy Score: 83.97%
    Precision Score: 83%
    Recall Score: 84%

  • Support Vector Machines (SVMs)

    • Confusion matrix:

    Predicted Income <=50K Predicted Income >50K
    Actual Income <=50K 5236 406
    Actual Income >50K 821 1078
    Accuracy Score: 83.72%
    Precision Score: 83%
    Recall Score: 84%

  • Random Forest

    • Confusion matrix:

    Predicted Income <=50K Predicted Income >50K
    Actual Income <=50K 5089 553
    Actual Income >50K 801 1098
    Accuracy Score: 82.04%
    Precision Score: 81%
    Recall Score: 82%

3. Training the Models

Using Scikit-learn's train_test_split module to split 75% of features (X) and target (y) data into training data and 25% for testing data.

4. Evaluating the Models

Using 25% testing data to evaluate the models. Comparing the actual result and the predicting result and calculating the confusion matrix and accuracy score for each model.

5. Determining the Model for predicting income value

To campare, the logistic regression model scored the highest in terms of accuracy score (83.97%), which was slightly better than the Support Vector Machine (83.72%) or Random Forest model (82.04%). Given this provided the highest level of accuracy, this model was used to develop our interactive income predictor built into our web page.
